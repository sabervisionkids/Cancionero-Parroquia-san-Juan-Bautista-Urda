# Cancionero Parroquia San Juan Bautista (Urda)

Aplicación web desarrollada para consultar el cancionero de la Parroquia San Juan Bautista de Urda de forma rápida, sencilla y completamente offline.

## Características

- Consulta rápida de canciones.
- Funcionamiento sin conexión a Internet.
- Compatible con ordenador, tablet y móvil.
- Búsqueda sencilla.
- Visualización de acordes integrada.
- Pensado para el uso en celebraciones, ensayos y reuniones parroquiales.

## Uso

Este repositorio se publica para facilitar el uso del cancionero por parte de la comunidad.

Puedes utilizar la aplicación como usuario final, pero el código fuente no puede copiarse, modificarse, redistribuirse ni utilizarse para desarrollar otros proyectos sin autorización expresa del autor.
